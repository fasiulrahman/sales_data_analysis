 #Sales Data Analysis using Python

## Project Overview
This project analyzes sales data using Python to identify sales trends, calculate total revenue, and visualize product-wise performance.

The project demonstrates basic data analysis and visualization techniques used in data science.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Dataset
The dataset contains the following columns:

- Date
- Product
- Sales
- Profit

## Features
- Load and analyze sales dataset
- Calculate total sales and profit
- Perform product-wise sales analysis
- Create data visualizations using bar charts

## Project Structure
sales_data_analysis  
│  
├── sales_data.csv  
├── sales_analysis.ipynb  
└── README.md  

## Output
The project generates insights about:
- Total sales
- Product-wise performance
- Sales visualization graphs
